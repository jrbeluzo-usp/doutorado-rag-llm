[DEFINITION]Uma **base de dados** é um conjunto estruturado de informações organizadas de forma a permitir seu armazenamento, recuperação e manipulação eficiente. No contexto de **dados estruturados**, uma base de dados é geralmente composta por tabelas relacionais com esquemas bem definidos, contendo linhas (registros) e colunas (atributos), como em bancos de dados SQL (ex.: MySQL, PostgreSQL, Oracle). Já no contexto de **dados não estruturados**, a base de dados pode armazenar informações sem um formato fixo ou predefinido, como documentos de texto, imagens, vídeos e logs, sendo gerenciadas por sistemas NoSQL (ex.: MongoDB, Elasticsearch) ou armazenamentos distribuídos para big data (ex.: Hadoop, Amazon S3).[/DEFINITION]
